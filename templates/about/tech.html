О проекте

Это учебный проект - платформа для блогов. Он готовился для курса Яндекса по бэкенд-разработке на Python. Проект собран на Django c использованием СУБД PostgreSQL, шаблонов HTML/CSS на основе стилей Bootstrap. Он работает как WSGI приложение на удаленном сервере в Docker-контейнерах. Все написанные модули покрыты автотестами.

Вот, что я знаю:

Python, SQL, Django, PostgreSQL, HTML, CSS, Bootstrap, REST API, JSON, OAuth, DRF, Django REST Framework, GIT, Unittest
